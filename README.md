# ZipTaxi

## Project Overview
 ZipTaxi는 개인택시 기사님들을 위한 수익 관리 앱입니다.

## Key Features
 1. 수익 기록 - 일별/월별 운행 수익 입력 (카드/현금/앱 결제 구분)                                                  
 2. 목표 관리 - 월별 목표 매출 설정 및 달성률 시각화                                                               
 3. AI 분석 - GPT-4o-mini 기반 맞춤형 운행 조언 제공

## Tech Stack
- **Frontend**: React Native (TypeScript)
- **Backend/Database**: Firebase (Firestore, Auth)
- **AI**: OpenAI API (GPT-4o-mini)
- **Chart**: react-native-gifted-charts
- **Navigation**: React Navigation (Bottom Tabs)

## Setup
1. Clone the repository
2. `npx react-native run-android` 실행
